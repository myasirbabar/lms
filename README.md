# Lab Management System
Console Based Lab management System for students and admin Build in C++ console

---------------
### How to Begin
- Open the main.cpp file in your editor. 
- Compile and run
- Happy Execution :)

### Details
- A student can log in from available system or log out of a system(Computer)
- Admin have access to all system records.
- Admin can search a student.
- Admin can log in or log out a student.
